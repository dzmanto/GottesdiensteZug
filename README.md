# GottesdiensteZug
Android app für reformierte Gottesdienste in Zug
Die Android app basiert auf der <a href="https://www.ref-zug.ch/">Homepage</a> der reformierten Kirche in Zug. Dort findet sich eine Liste mit <a href="https://www.ref-zug.ch/zug-menzingen-walchwil/gottesdienste/">reformierten Gottesdiensten</a> in Zug.
Die App holt jene Informationen über Gottesdienste aufs Natel und stellt diese übersichtlich dar. Die App fokusiert sich auf das Wesentliche, das heisst auf Termine und Orte der Gottesdienste. Es können verschiedene Bezirke ausgewählt werden.
Die App ist ein privates Projekt. Sie ist nicht in Zusammenarbeit mit der Büro entstanden, welches für das Webdesign der <a href="https://www.ref-zug.ch/">Homepage</a> der reformierten Kirche verantwortlich ist.
